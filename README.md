# Mosca E-commerce

Um e-commerce moderno e elegante construído com Next.js, TypeScript e Tailwind CSS.

## Tecnologias

- **Next.js 14** - Framework React com Server Components
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Framework CSS de utilidade-first
- **Radix UI** - Componentes acessíveis
- **Lucide React** - Ícones modernos

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) no seu navegador

## Estrutura do projeto

- `src/app/` - Páginas e layouts do Next.js
- `src/components/ui/` - Componentes reutilizáveis
- `src/lib/` - Utilidades e funções compartilhadas
